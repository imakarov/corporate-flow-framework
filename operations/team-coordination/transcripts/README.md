Original transcripts from team calls (any recording service).

## Guidelines

### File Naming
- **Preserve original format**: `YYYY-MM-DDTHH:MM:SS-TZ-Meeting_Name.txt`
- **Never modify** original transcripts
- Let recording service name the file

### What Goes Here
- Raw call transcripts
- Original recordings metadata
- Unedited meeting captures

### Processing Transcripts
1. Original transcript stays here (never modified)
2. Create summary in `../notes/YYYY-MM-DD-[topic].md`
3. Include metadata linking back to this transcript
4. Extract action items to `../tracking/weekly-plan.md`

### Language Notes
- Keep originals in native language
- Summaries can be in preferred language

## Example Workflow

```bash
# 1. Transcript arrives from recording service
transcripts/2025-10-22T10:00:00-04:00-Daily_Standup.txt

# 2. Create summary
notes/2025-10-22-daily-standup.md

# 3. In summary, link back:
Sources:
- operations/team-coordination/transcripts/2025-10-22T10:00:00-04:00-Daily_Standup.txt
```
